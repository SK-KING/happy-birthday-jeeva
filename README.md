## 🛠️ How to Use

### **Option 1: GitHub Pages (Automatic Hosting)**
1. Go to repository settings: `https://github.com/SK-KING/happy-birthday-jeeva/settings`
2. Scroll to "GitHub Pages" section
3. Select branch: `main`
4. Source: `/ (root)`
5. Click Save
6. Your site will be live at: `https://SK-KING.github.io/happy-birthday-jeeva/`

### **Option 2: Local Testing**
1. Clone the repository:
```bash
git clone https://github.com/SK-KING/happy-birthday-jeeva.git
cd happy-birthday-jeeva
```

2. Open `index.html` in your browser or use a local server:
```bash
# Using Python 3
python -m http.server 8000

# Or using Node.js
npx http-server
```

3. Open `http://localhost:8000` in your browser

## 🎨 Customization

### **Change Birthday Message**
Edit `index.html` search for "wish-text" section:
```html
<p class="wish-text">
    ✨ Your custom message here! ✨
</p>
```

### **Add Your Photos to Gallery**
Edit the photo gallery section in `index.html`:
```html
<div class="photo-item">
    <img src="your-photo.jpg" alt="Memory">
</div>
```

### **Change Colors**
Edit `style.css` root variables:
```css
:root {
    --primary: #FF6B9D;      /* Pink */
    --secondary: #FFA502;    /* Orange */
    --tertiary: #00B4DB;     /* Blue */
    --accent: #FFD700;       /* Gold */
}
```

### **Change Birthday Song**
Edit `index.html` search for audio tag:
```html
<audio id="birthdayAudio">
    <source src="YOUR_SONG_URL.mp3" type="audio/mpeg">
</audio>
```

## 🐛 Troubleshooting

### **Animations not playing?**
- Clear browser cache (Ctrl+Shift+Delete)
- Try a different browser
- Check if JavaScript is enabled

### **Sound not working?**
- Check if volume is enabled in browser
- Try enabling in settings
- Some browsers require user interaction first

### **Page looks broken?**
- Clear cache and refresh
- Try in incognito/private mode
- Update your browser

### **Music file not loading?**
- Check internet connection
- Verify audio file URL is correct
- Try a different audio source
